2- school managment application

The resources available on Drive following the link gestion schoolaire resources or on Github following the link gestion schoolaire resources

the school mangment app its a large project that contains frontend and backend with difrents technologies we walk throw on ach one of those the steps to run that project will splited on tow part but first

  • Download the zip file from links above
  • Extract the zip file.

a. run the backend server

java jakarta EE based application work throw apis with maven build tool and deployed on embed tomact serevr

  • open the project extracted folder in your IDE
  • open console window
  • navigate to server folder inside the project folder
  • your console prompt should look like : C:\\Users\your-user-name\path\to\extracted\folder\School-manage-app\server
  • run the command mvn clean install in console window
  • you should see build success
  • when installing dependency done run the main class with command mvn exec:java -Dexec.mainClass="com.example.Server"
  • when see Server running at: http://localhost:8080/app on the console congrates the backend server is running

NOTE : if you don't use eclips IDE you should have maven installed on your machine you can download it

b. run the frontend server

ReactJs based application with vite build tool and deployed on vite embed bundler

NOTE: you need to install NodeJs with npm from here

  • open the project extracted folder in your IDE
  • open console window
  • navigate to admin-panel folder inside the project folder
  • your console prompt should look like : C:\\Users\your-user-name\path\to\extracted\folder\School-manage-app\admin-panel
  • run the command npm install in console window
  • when installing dependency done run the command to start frontend server npm run dev
  • when see âžœ Local: http://localhost:5173/ on the console congrates the frontend server is running
  • visit http://localhost:5173/ on browser and enjoy

c. alternative way to test the backend server

you can use postman or soapUI or any Api testing tool to test the backend server apis

  • open your testing tool application
  • import requestes json file located in C:\\Users\your-user-name\path\to\extracted\folder\School-manage-app\School-managment-app.postman_collection.json
  • choose an endpoint folder
  • pick one of available requeste (all,one,create,update,delate)
  • press the send button
  • you sould recive a responce from server with serval data from java application
